# flipkart-ecommrance
# 🛒 Flipkart E-Commerce Clone

A responsive **Flipkart-inspired E-Commerce website** built using **HTML, CSS, JavaScript, Node.js, and Express.js**. This project recreates the basic shopping experience with multiple product categories and user authentication pages.

---

## 🚀 Features

- 🏠 Home Page
- 👕 Fashion Section
- 📱 Mobiles Section
- 💻 Electronics Section
- 📺 TV Section
- 🛋 Furniture Section
- 📦 Products Page
- 🔐 User Login
- 📝 User Signup
- ⚡ Responsive Design
- 🌐 Express.js Backend

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/flipkart-ecommerce.git
```

### Navigate into Project

```bash
cd flipkart-ecommerce
```

### Install Dependencies

```bash
npm install
```

### Start Server

```bash
node server.js
```

or

```bash
nodemon server.js
```
